Run HelloWorld using JavaFX 11

Download an appropriate JavaFX runtime for your operating system and unzip them to a desired location.

Add an environment variable pointing to the lib directory of the runtime: 


export PATH_TO_FX=/path/to/javafx-sdk-11/lib  #Linux and macOS
set PATH_TO_FX="path\to\javafx-sdk-11\lib"  &::Windows

You can now compile and run JavaFX 11 applications from the command line using the JavaFX runtime.

Compile the application (e.g. use HelloFX.java) using: 


javac --module-path $PATH_TO_FX --add-modules=javafx.controls HelloFX.java  #Linux and macOS
javac --module-path %PATH_TO_FX% --add-modules=javafx.controls HelloFX.java  &::Windows

Make sure to add the required modules, keeping into account transitive dependencies are automatically resolved (there is no need to add javafx.graphics module, since it is transitively required by the javafx.controls module).

Next, run the application using: 


java --module-path $PATH_TO_FX --add-modules=javafx.controls HelloFX  #Linux and macOS
java --module-path %PATH_TO_FX% --add-modules=javafx.controls HelloFX  &::Windows
